
# AleksOS BETA   консоль на базе Nofrendo
AleksOS — это мой экспериментальный проект, основанный на эмуляторе NES **Nofrendo** 
и адаптированный под микроконтроллеры ESP32-2432S028 или дешовый жолтый дисплей Cheap Yellow Display

## 📜 Лицензия
Проект распространяется под лицензией **GNU GPL v2.0**, в соответствии с лицензией оригинального Nofrendo.

## Модификации Nofrendo

В проекте используется эмулятор Nofrendo , 
который был модифицирован для работы на моей аппаратной платформе.

Основные изменения:
- адаптация под ESP32;
- переработка работы с памятью;
- оптимизация рендеринга под ограниченные ресурсы;
- добавлены собственные функции и исправления.

Все изменения распространяются в соответствии с лицензией GNU GPL v2.0.

## ⚠️ Важное предупреждение по установке PSRAM

Плата была модифицирована: добавлена внешняя микросхема PSRAM 
APS6404L‑3SQR‑SN. Для связи с PSRAM используются выводы, которые на 
оригинальной плате были задействованы под RGB‑светодиод.

⚠️ ВАЖНО:
Если припаять PSRAM ДО прошивки платы — микросхема может сгореть.

Причина:
На линиях RGB‑светодиода при старте подаётся напряжение. 
Так как эти же линии используются для интерфейса PSRAM, 
напряжение попадает прямо на выводы APS6404L‑3SQR‑SN, что приводит 
к её повреждению или полному выходу из строя.

Правильный порядок установки:
1. Сначала прошить плату.
2. Затем припаять микросхему APS6404L‑3SQR‑SN.
3. После этого включать питание.


## Используемая PSRAM

Проект рассчитан на работу с внешней микросхемой PSRAM **APS6404L‑3SQR‑SN** 
(8 MB QSPI PSRAM). Прошивка и вся логика работы памяти адаптированы 
именно под эту модель.

Использование других вариантов PSRAM не гарантируется.

Добавление PSRAM
4 МБ PSRAM можно добавить к стоимости установленного RGB-светодиода (ну, 2/3). На мой взгляд, дополнительная память будет полезнее для мультимедийных приложений, для которых плата предназначена, чем один светодиод, установленный на задней панели.
Для дополнительного чипа флэш-памяти предусмотрен SOIC8. Мы можем использовать его для PSRAM, так как он использует большинство контактов с флэшем — кроме двух:
 <img width="4000" height="2250" alt="cyd_PSRAM_mod" src="https://github.com/user-attachments/assets/13a14e45-822b-409a-b7bd-12c2a5f268f4" />
такт (должен быть GPIO17)
выбор чипа (должен быть GPIO16)
Оба GPO используются RGB-светодиодом. После удаления мы можем вырезать существующие дорожки для двух вышеуказанных сигналов и подключить их к соответствующим светодиодным площадкам. Оба контакта будут иметь резисторы по 1k. Для CS это обязательно, для CLK это не имеет значения, разве что потребление тока немного больше. R16 можно удалить при необходимости.
Мод PSRAM
Встроенный светодиод может быть удобен при отладке, красная часть RGB-светодиода (средние контакты) может использоваться для добавления светодиода 1206, как показано на фото. Этот светодиод доступен на GPIO4.

Подходящие чипы PSRAM:
APS6404L-3SQR-SN - 64 Мбит (8 МБ) версии, ESP отображает только половину. (Mouser# 878-APS6404L-3SQR-SN)
APS1604M-3SQR-SN — версия на 16 Мбит (2 МБ), вероятно, достаточно для аудиостриминга и декодеров. (Mouser# 878-APS1604M-3SQR-SN)
Важно! - это должна быть версия с 3SQR. Модели без 3 — это 1.8V. В CYD это не подойдёт.
